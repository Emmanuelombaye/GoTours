This is a Vite + React + Tailwind + Framer Motion frontend prototype for a clean, Airbnb-style villa listings homepage.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build

```bash
npm run build
npm run preview
```

Notes
 
 API fallback
 - The app will fetch `/api/villas` if available. When developing locally it uses the local `src/data/villas.json`.
 
 Run the app
 
 ```bash
 npm install
 npm run dev
 ```

Deployment

- Build a production bundle:

```bash
npm run build
```

- Preview the production build locally:

```bash
npm run preview
```

- To deploy to GitHub Pages, build then push the `dist` folder to the `gh-pages` branch or use a static host (Netlify, Vercel). Example with `gh-pages` package is left as an exercise.
